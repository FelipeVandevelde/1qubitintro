# Qubit: uma introdução visual

Estrutura inicial para transformar o repositório `livroqubit/1qubitintro` em uma página pública com GitHub Pages.

## Estrutura sugerida

- `index.html` — página principal
- `codigos/` — exemplos e implementações
- `erratas/` — correções por edição e capítulo
- `materiais/` — leituras, figuras, slides e extras
- `assets/` — CSS, imagens e arquivos estáticos
- `404.html` — página de erro personalizada
- `.nojekyll` — evita processamento desnecessário do Jekyll

## Como publicar no GitHub Pages

1. Envie estes arquivos para a branch `main` do repositório.
2. No GitHub, abra `Settings` > `Pages`.
3. Em `Build and deployment`, escolha `Deploy from a branch`.
4. Selecione a branch `main` e a pasta `/(root)`.
5. Salve.
6. O endereço padrão do site será:
   `https://livroqubit.github.io/1qubitintro/`

## Organização editorial recomendada

### Códigos
- um diretório por capítulo ou tema
- cada exemplo com objetivo, dependências e instruções de execução
- notebooks e scripts com nomes consistentes

### Erratas
- por edição
- por capítulo
- com página, trecho e correção
- com data de atualização

### Materiais extras
- leituras comentadas
- figuras em alta resolução
- slides
- links de apoio

## Observação

Este modelo usa HTML e CSS puros. Isso reduz a complexidade inicial e facilita a manutenção.
